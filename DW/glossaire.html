Glossaire

HTML
definition : HTML (HyperText Markup Language) est le langage de balisage utilisé pour structurer le contenu des pages web.
CSS
definition : CSS (Cascading Style Sheets) est un langage utilisé pour décrire la présentation et le style des documents HTML.
JavaScript
definition : JavaScript est un langage de programmation qui permet d’ajouter de l’interactivité et des comportements dynamiques aux pages web.
URL
definition : URL (Uniform Resource Locator) est l’adresse web qui permet d’accéder à une ressource sur Internet, comme une page, une image ou un fichier.
